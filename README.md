# easy-json-db

Simple to use JSON file system meant for discord bots. It was designed and built for small and prototype applications.

# Breakdown

### Features

- Small and lightweight.
- Typescript typings.
- Easy to edit raw data.

### Drawbacks

- Not very performant
- Minimal features
- You pledge alliance to the JSON overlords.
